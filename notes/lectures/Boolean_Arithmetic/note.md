# 单bit加法

# 16-bit adder
利用15个full adder以及一个 half adder

# 负数表示
把第一位用在符号上。但这个在表示0的时候有比较大的问题。

人们一般使用$2^n-x$来表示$-x$.

$2^n-x=1+(2^n-1)-x=1+(1111111)-x$
而用1111111减去x等价于对x取反。

但是函数f很多时候是由控制位构成的。
